## Question

아래 예제와 같이 고양이를 출력하시오.

### Input

> 입력 없음

### Output

고양이를 출력한다.

### Example Input

> 입력 없음

### Example Output

```
\    /\
 )  ( ')
(  /  )
 \(__)|
```

## Solution

```js
console.log(`\\    /\\\n )  ( \')\n(  /  )\n \\(__)|`);
```

백틱(`)을 사용해 출력했다.

### 백슬래시 사용 출력 문법

> `\\` : 역슬래시 두번은 \ 역슬래시를 한번을 출력  
>  `\n` : 줄바꿈  
>  `\'` : 작은 따옴표
